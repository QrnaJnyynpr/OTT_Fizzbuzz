FIZZBUZZ.

The classic programming test.

If you haven't heard of the Fizzbuzz Test before, here are the rules:

Write a program that prints the numbers 1 to 100. For every number that is a multiple of 3, replace it with "Fizz". For every number that is a multiple of 5, replace it with "Buzz". To make it a bit more complicated, for every number that is both a multiple of 3 and 5, replace it with "Fizzbuzz".
